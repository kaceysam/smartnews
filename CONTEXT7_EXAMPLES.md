## Tips for Better Results

1. **Be Specific**: Include the exact library version if you know it
   ```
   Show me React 18.2 useState patterns. use context7
   ```

2. **Include Your Stack**: Mention related technologies
   ```
   How do I use Supabase with Next.js 14 App Router and TypeScript? use context7
   ```

3. **Ask for Best Practices**: Request modern patterns
   ```
   What are the current best practices for React state management in 2024? use context7
   ```

4. **Request Examples**: Ask for complete, working examples
   ```
   Show me a complete example of a Next.js API route that validates input and returns JSON. use context7
   ```

5. **Combine Concepts**: Ask about integrating multiple libraries
   ```
   How do I use React Query with Supabase for real-time data fetching in Next.js? use context7
   ```
